# Sheryians Coding School — Clone

A responsive landing page clone of the **Sheryians Coding School** website, built using pure HTML, CSS, and vanilla JavaScript.

## 🚀 Live Demo

[View Live Site](https://pritivish07025.github.io/sheryians-coding-school-clone/)

## 📁 Project Structure

```
sheryians-coding-school-clone/
│
├── index.html      # Main HTML file with all sections
├── style.css       # All styling + responsive media queries
└── README.md       # Project documentation
```

## ✨ Sections Included

- **Navbar** — Logo, navigation links, sign-in button
- **Hero Section** — Course title, pricing, CTA buttons, course thumbnail with play button
- **Get Placed** — Gradient clipped heading with stats overview
- **Stats Section** — Hours, lectures, and questions cards with glow effect
- **Comparison Section** — Sheryians vs other coding platforms
- **Curriculum** — Course prerequisites card + interactive module accordion (5 modules)
- **Certification** — Certificate of completion preview card
- **FAQ Section** — Accordion-style frequently asked questions
- **CTA Section** — "Buy Now" call-to-action with floating background cards
- **Footer** — Giant outline logo text, social links, and contact info

## 🛠️ Built With

- **HTML5** — Semantic page structure
- **CSS3** — Flexbox, Grid, animations, gradients, `clamp()` for responsive typography

## 📱 Responsive Design


Media queries handle layout stacking, font scaling, and hiding decorative elements on smaller screens.

## ⚙️ Features

- ✅ Interactive accordion for course modules (click to expand/collapse)
- ✅ Single-open FAQ accordion
- ✅ CSS gradient & image-clipped text effects
- ✅ Smooth hover transitions on buttons and links
- ✅ Fully responsive grid layouts

## 🖥️ How to Run Locally

1. Clone or download this repository
2. Open `index.html` in any browser

No build tools, no dependencies — just open and go.

## 🌐 Deployment (GitHub Pages)

1. Push code to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select branch `main` and folder `/ (root)`
4. Save — your site will be live at `https://<username>.github.io/<repo-name>`
